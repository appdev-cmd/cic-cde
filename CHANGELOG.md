# Changelog

All notable changes to CDE CIC are documented here.

## [0.2.0] - 2026-06-08

### Added

- Complete CDE frontend prototype with React 19 + TypeScript + Vite.
- BIM viewer integration using ThatOpen Components (IFC loading, element
  selection, property inspector).
- ISO 19650 document explorer with status workflow (WIP, SHARED, PUBLISHED,
  ARCHIVED).
- 4D scheduling tab with Gantt chart visualization.
- AI assistant drawer powered by Google Gemini.
- GeoBIM/GIS dashboard with map integration.
- Project overview with global KPIs.
- Dark mode with localStorage persistence.
- Material Design 3 design system with custom color tokens.

### Harness

- Initialized repository-harness with durable layer (SQLite + Rust CLI).
- Created 8 user stories (US-001 through US-005, US-FE-001 through US-FE-004).
- Recorded 4 architecture decisions (0006 through 0009).

## [0.1.0] - 2026-06-07

### Added

- Project initialization with technical evaluation documents.
- Detailed implementation plan for 3 subsystems.
- Feasibility study report.
- Legal and regulatory reference documents.
